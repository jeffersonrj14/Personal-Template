## Images

uses [Next.js Image](https://nextjs.org/docs/basic-features/image-optimization) in MDX for seamless image rendering:

```
<Image
  src="/Opengraph-Images.png"
  alt="OpenGraph image"
  width={640}
  height={500}
/>
```

Renders:

<Image
  src="/Opengraph-Images.png"
  alt="OpenGraph image"
  width={640}
  height={500}
/>

## Image Grid

uses a custom [image grid component](https://github.com/1msirius/Nextfolio/blob/main/app/components/image-grid.tsx) to display image galleries.

```
<ImageGrid
  columns={3} // Accepts 2, 3, or 4 columns
  images={[
    { src: "/Opengraph-Images.png", alt: "Photo1", href: "#" }, // 'href' is optional
    { src: "/Opengraph-Images.png", alt: "Photo2", href: "#" },
    { src: "/Opengraph-Images.png", alt: "Photo3"},
  ]}
/>
```

Renders:

<ImageGrid
  columns={3}
  images={[
    { src: "/Opengraph-Images.png", alt: "Photo1", href: "#" },
    { src: "/Opengraph-Images.png", alt: "Photo2", href: "#" },
    { src: "/Opengraph-Images.png", alt: "Photo3" },
  ]}
/>

## Embeds

### Tweets

uses [react-tweet](https://react-tweet.vercel.app/) to embed tweets in MDX posts.

```
<StaticTweet id="1987177773107163546" />
```

Renders:

<StaticTweet id="1987177773107163546" />

### YouTube Videos

Uses [react-youtube](https://www.npmjs.com/package/react-youtube) to embed YouTube videos in MDX posts.

```
<YouTube videoId="mwJGZ8YU7o4" />
```

Renders:

<YouTube videoId="mwJGZ8YU7o4" />

### Captions

uses [react-wrap-balancer](https://react-wrap-balancer.vercel.app/) to evenly balance captions in MDX posts.

```
<Caption>
  This site is built using the [Nextfolio template](https://github.com/3p5ilon/Nextfolio), customized to fit my workflow as my personal go-to template
</Caption>
```

Renders:

<Caption>
  This site is built using the [Nextfolio template](https://github.com/3p5ilon/Nextfolio), customized to fit my workflow as my personal go-to template
</Caption>

## Tables

## Math Expressions

render mathematical expressions in MDX posts using [KaTeX](https://katex.org/) . Simply wrap your expression in **$** to include KaTeX within your MDX content.

```
$
\int_0^\infty e^{-x^2} dx = \frac{\sqrt{\pi}}{2}
$

$
\sigma = \sqrt{\frac{1}{N}\sum_{i=1}^{N}(x_i - \mu)^2}
$
```

Renders:

$
\int_0^\infty e^{-x^2} dx = \frac{\sqrt{\pi}}{2}
$

$
\sigma = \sqrt{\frac{1}{N}\sum_{i=1}^{N}(x_i - \mu)^2}
$

Don't push production on Friday
</Callout>
